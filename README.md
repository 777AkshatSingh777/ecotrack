# EcoTrack: Personal Carbon Footprint Tracker 🌱

EcoTrack is a beginner-friendly web application designed to help individuals calculate, understand, and reduce their daily carbon footprint through interactive quizzes and actionable challenges.

## 🌟 Features
- **Interactive Quiz:** Simple questions about transportation, diet, and energy usage.
- **Dynamic Score:** Instantly calculates carbon impact using standard environmental formulas.
- **Daily Badges:** Users unlock digital badges (e.g., "Eco Warrior") when completing challenges.
- **Responsive Design:** Works beautifully on both laptops and mobile phones.

## 🛠️ Tech Stack
- **HTML5:** For page structure.
- **CSS3:** Custom styling and smooth animations (no heavy frameworks needed!).
- **JavaScript (ES6):** For calculation logic and updating the score in real-time.
